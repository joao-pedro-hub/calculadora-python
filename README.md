#  Calculadora Simples em Python

Projeto desenvolvido em Python com o objetivo de treinar lógica de programação, uso de funções, estruturas condicionais e repetição.

A aplicação funciona no terminal e permite ao usuário realizar operações matemáticas básicas por meio de um menu interativo.

##  Funcionalidades

- Soma de dois números
- Subtração de dois números
- Multiplicação
- Divisão com validação de divisão por zero
- Menu interativo
- Opção de sair do programa

##  Conceitos praticados

- Funções em Python
- Estrutura de repetição (`while`)
- Estrutura condicional (`match / case`)
- Entrada de dados com `input()`
- Tratamento de erros lógicos
- Organização de código

##  Tecnologias

- Python 3

##  Como executar

1. Certifique-se de ter o Python instalado (versão 3.10 ou superior).
2. Execute o arquivo no terminal:

```bash
python calculadora.py
